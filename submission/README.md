# Submission

Before you push and share your repo link, this folder must contain your prompt log.

1. Copy your log file into this folder:
   - macOS / Linux: `cp ~/hackerrank_buildathon/log.txt submission/log.txt`
   - Windows (PowerShell): `Copy-Item "$env:USERPROFILE\hackerrank_buildathon\log.txt" submission\log.txt`
2. Make sure your code is committed in `code/` and your predictions are in `support_tickets/output.csv`.
3. Commit and push everything, then send us your GitHub repo link.

That single link is your whole submission: code, output, and prompt log.
